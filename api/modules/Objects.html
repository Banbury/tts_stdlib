Functions

collect (o, fn)
Executes the callback for every object in a table and collects the return value in a new table.

Objects.collect(getAllObjects(), function(o) return o.guid end)

This call returns the guids of all objects as a table.

Parameters:

  • o Table of objects
  • fn Callback function with one parameter. The return value will be collected.
execute (o, fn)
Executes a function on all given objects. The callback function must have a single parameter. The function will be called once for every object, with the object as an argument.

Objects.execute(objects, function(o) o.highlightOn({1, 1, 0}, 5) end)

Parameters:

  • o Table of objects
  • fn Callback function with one parameter and no return value.
find (o, fn)
Find objects based on the return value of a function. The find function expects a table of objects and a function with a single parameter. This function is called once for every object, with the object as an argument. The function needs to return true for all objects, that should be found.

Objects.find(getAllObjects(), function(o) return o.name!="BlockSquare" end)

Parameters:

  • o Table of objects
  • fn Callback function with one parameter. Must return true or false.

Returns:

    Returns a table of objects, selected by the callback function.
getAllHeldByColor (color)
Returns all objects held by the player with the given color.

Parameters:

  • color Color of the player as string.

Returns:

    Returns a table of objects.

Parameters:

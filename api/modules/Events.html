Module Events

Functions for sending events between objects.

EventHandler must be included in Global.

#include !/events/eventhandler

Events can be included in any object except Global.

#include !/events/events

Functions

EventHandler.register (event, fn)

Registers a function as an event handler. In order to receive an event with a certain name, a function has to be registered as an event handler. If an object sends an event with the same name, this function will be called. There can only be one event handler per event. The event handler function has one parameter. This parameter receives a table of arguments provided by the sender. The table always contains

  • guid - the guid of the sender
  • event - the name of the event

Parameters:

  • event Name of the event as a string.
  • fn The function to be called, if the event is sent.
EventHandler.send (event, args)
Sends an event to all subscribers. This function is used to send events from Global.

Parameters:

  • event Name of the event as a string.
  • args Table of arguments to be sent to sunscribers.
EventHandler.start ()
Starts the event loop. This function has to be called before any events can be processed.
EventHandler.stop ()
Stops the event loop.
EventHandler.unregister (event)
Removes an event handler function for an event. The object will receive no further events of this name. Throws an error, if the event has not been registered before.

Parameters:

  • event The name of an event as a string.

If an object sends an event with the same name, this function will be called. There can only be one event handler per event. The event handler function has one parameter. Event.send (event, args)
Sends an event to all subscribers. This function is used to send events from Global.

Parameters:

  • event Name of the event as a string.
  • args Table of arguments to be sent to sunscribers.
Event.subscribe ()
Subscribes the calling object with the global event loop. This function has to be called in an object, before any events can be received. Global is subscribed automatically.
Event.unregister (event)
Removes an event handler function for an event. The object will receive no further events of this name. Throws an error, if the event has not been registered before.

Parameters:

  • event The name of an event as a string.
Event.unsubscribe ()
Unsubscribes the calling object from the global event loop. The calling object receives no event at all anymore.
